# SCT_DS_3
## SkillCraft Technology Data Science Internship- Task3: Decision Tree Classifier using Bank Marketing Dataset
## Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behaviorial data using the Bank Marketing Dataset.
## Dataset 
-Bank Marketing Dataset(UCI Machine Learning Repository)
## Tools & Libraries Used
-python
-pandas
-numpy
-matplotlib
-seaborn
-scikit-learn
-google colab
## Steps Performed
-Imported and explored the dataset.
-Checked for missing values.
-Encoded categorical variables using LabelEncoder.
-Split the Dataset into training and testing sets.
-Built a Decision Tree Classifier.
-Evaluated the model using accuracy,confusion matrix,and classification report.
-Visualized the decision tree.
-Analyzed feature importance.
## Results
-Successfully trained a Decision Tree model to predict customer subscription.
-Evaluation the model's performance using standard classification metrics.
-Identified the most important features influencing predictions.
## Conclusion
This project demonstrates the implementation of a Decision Tree Classifier for a classification problem using the Bank Marketing Dataset.The model was trained,evaluated, and interpreted through visualization and feature importance analysis.
